BCA-204:System Analysis and Design

Unit-1: Importance of System Analysis and Design, Concept of System, Characteristics and types of System, System Development Life Cycle - Waterfall model, Prototype development strategy, Software Crisis, Audit Trail, Role and Attributes of System Analyst.

Unit-II: Sources of Project request, Project Selection, Preliminary Investigation, Feasibility Study, types of feasibility study, Cost and bebefit analysis. Systems Analysis, Requirements determination, Structured Analysis, System Requirements Specification(SRS), Analysis Tools, Data Dictionary, HIPO, Decision Tree, Decision Table, Warnier-Orr Diagram, ERD, Data Flow Design(DFD).

Unit-III: System Design, Design Process, constraints, Input and control Design, Validation checks, concept and types of Form, Form Design, Output Design, File Design: Types of File - Master File, Transaction File, Table / Reference File, Report File.

Unit-IV: Software Quality and Testing: Quality concept, Testing and Testing Objectives, Black Box and White Box testing, Unit testing, integration testing, system testing.

Unit-V: Hardware and Software Acquisition and criteria, system conversion and conversion plan, Site preparation, training, Maintenance and types of maintenance.

Unit-VI: Management Information System: Definition, objectives and role in organization and for managers. Information system at different levels of organisation and for managers. Information system at different levels of organisation/management - OAS, TPS, MIS, DSS. Components of MIS, Failure and success of MIS.
